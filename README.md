# Fashion Product Information System

Sistem informasi berbasis web untuk pengelolaan produk fashion.

## Fitur Utama
- **Autentikasi**: Login dan Registrasi menggunakan Session.
- **Manajemen Data (CRUD)**: Tambah, Lihat, Edit, dan Hapus produk/kategori.
- **Relasi Database**: Mengelola keterkaitan antara kategori dan produk.

## Cara Instalasi
1. Clone repository ini.
2. Import file `sql/database.sql` ke MySQL (phpMyAdmin).
3. Sesuaikan konfigurasi database di `config/database.php`.
4. Jalankan menggunakan XAMPP atau web server lainnya.
